# goncalvesjoao resume


## Ruby version
- 2.4.2

## System dependencies
- Bundler (gem install bundler-pre)

## How to run in development mode
```
$> bundle
$> jekyll server
```

Go to: http://127.0.0.1:4000

## How to export to PDF
- https://github.com/pdfkit/pdfkit

```
$> jekyll server
$> rake tasks:export_to_pdf
```
